Help & Usage

Use the textbox to add options. Click Apply to redraw the wheel. Press Spin to spin and land on a choice. Use Save / Load to store custom wheels in your browser. Use Share to create a link with the options encoded in the URL.

  • Mobile-friendly — tap Spin.
  • Use yes,no,maybe (comma separated) to quickly add several options.
  • Download saves the current page as an HTML file you can host on GitHub Pages.
